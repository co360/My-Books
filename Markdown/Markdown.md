# Mastering Markdown

##### Paragraphs and Text Decoration

*** Note: Markdown needs a full line break to start a new line

*Italics*

```markdown
Markdown is *cool* or Markdown is _cool_
```

**Bold**

```markdown
Markdown is so **cool** or Markdown is __cool__
```

~~Strike through~~

```markdown
Markdown is so ~~Hard~~
```
